Introduction to HTML

HTML stands for Hyper Text Markup Language. It is used to design web pages using markup language. It is combination of Hypertext and Markup language. Hypertext defines the link between the web pages. Markup language is used to define the text document.

Introduction to CSS

Cascading Style Sheets, fondly referred to as CSS, is a simple design language intended to simplify the process of making web pages presentable. CSS handles the look and feel part of a web page. Using CSS, you can control the color of the text, the style of fonts.

Introduction to JavaScript

JavaScript is a very powerful client-side scripting language. JavaScript is used mainly for enhancing the interaction of a user with the webpage. In other words, you can make your webpage more lively and interactive, with the help of JavaScript.
